## 🎬 开场（10-20 秒）

今天我们讲 Prompt 工程 里的「JSON 与输出控制」。这一课不要死记硬背，先抓住一个小白也能懂的主线：今日主题：让 AI 变成可编程的零件。

这支视频必须覆盖当天所有题目，一共 10 个考点。看完以后，你不仅知道答案，还能说出为什么。

## 📖 主体讲稿

### Slide 1 — 先建立今天的直觉

把今天的内容想成一个工具箱。遇到题目时，先问三件事：它在解决什么问题？输入是什么？输出应该长什么样？只要这三件事想清楚，选择题、连线题和开放实战题都会变简单。

### Slide 2 — 考点 1-4：逐题白话拆解

- 当题目问「价值：结构化输出（如 JSON）相比自由文本的最大优势是？」时，正确理解是：可以被程序直接解析和处理，接入自动化流水线。
- 当题目问「场景：你需要让 AI 批量分析 1000 条评论的情感。输出应该用什么格式？」时，正确理解是：JSON 数组，每条包含 id、text、sentiment 字段。
- 当题目问「API：OpenAI API 中强制 AI 输出合法 JSON 的参数叫 response_format，类型设为 ___。」时，正确理解是：json_object。
- 当题目问「技巧：以下哪种 Prompt 最能确保 AI 输出合法 JSON？」时，正确理解是：请严格按照以下 JSON 模板输出，不要添加任何解释文字：
{"name": "string", "age": "number", "city": "string"}。

讲的时候提醒学员：不要只背关键词，要把关键词放回真实场景里。题目一换皮，场景理解才是拿分关键。

### Slide 3 — 考点 5-8：逐题白话拆解

- 当题目问「Debug：AI 输出了 JSON，但在 JSON 前面加了一句「以下是结果：」，导致你的程序解析失败。怎么修？」时，正确理解是：在 Prompt 中明确加上「只输出 JSON 对象本身，不要包含任何其他文字或解释」。
- 连线题要记成配对表：JSON = 接入前端/后端程序 API；Markdown 表格 = 在文档中直观对比；CSV = 批量导入 Excel 或数据库；YAML = 生成配置文件。
- 开放实战要讲清楚：把自然语言改成 JSON 输出要求：把下面这段需求改写成一个要求 AI 输出 JSON 的 Prompt，并定义至少 3 个字段：

「请帮我从一段餐厅评论中提取菜品名、评分、是否推荐。」 评分时看这些点：是否明确要求 JSON 输出（且只输出 JSON）；是否定义了至少 3 个字段；字段类型是否清晰（字符串 / 数字 / 布尔）；是否说明字段缺失时怎么处理。
- 开放实战要讲清楚：去掉解释性文字：下面是一个返回带解释的 Prompt：

「请把下面这段评论分类为「正面 / 负面 / 中性」，并解释你的判断理由。」

请你改写它，让 AI 只返回纯 JSON（包含 label 和 confidence），方便程序解析。 评分时看这些点：是否要求只输出 JSON、不输出解释；是否定义了清晰字段（如 label / confidence）；是否说明 label 的取值集合；是否考虑“无法判断”的情况。

讲的时候提醒学员：不要只背关键词，要把关键词放回真实场景里。题目一换皮，场景理解才是拿分关键。

### Slide 4 — 考点 9-10：逐题白话拆解

- 开放实战要讲清楚：设计你自己的 JSON 提取任务：挑一个你真实想自动化的小任务（如：从招聘 JD 里提取关键信息、从订单短信里提取金额和店家），写出对应的 JSON 提取 Prompt。 评分时看这些点：是否选了真实场景；是否定义了至少 3 个字段；是否处理“信息缺失”的情况；是否要求纯 JSON；是否能直接给 AI 使用。
- 开放实战要讲清楚：综合大题：请写一份完整的「用户反馈结构化提取」Prompt，准备给后端程序直接调用。要求：
1) 明确指令和角色；
2) 完整的 JSON Schema（至少 5 个字段，包含字符串 / 数字 / 布尔 / 数组各一种）；
3) 字段缺失时的默认值约定；
4) “只返回 JSON”的强约束；
5) 一段示例输入和示例输出，证明 AI 能照做。 评分时看这些点：是否覆盖 5 项；JSON 结构是否合理且字段类型多样；是否处理缺失值；是否给出清晰的输入输出示例；是否能直接接到程序里用。

讲的时候提醒学员：不要只背关键词，要把关键词放回真实场景里。题目一换皮，场景理解才是拿分关键。

### Slide 5 — 小白最容易混淆的点

- 如果题目问“概念”，先给一句人话定义，再补英文术语。
- 如果题目问“场景”，先判断任务类型，再选工具或 Prompt 策略。
- 如果题目问“Debug”，先找根因，不要直接堆更多功能。
- 如果题目问“Boss 综合”，答案要有结构：目标、步骤、格式、边界、检查。

### Slide 6 — 本课实战模板

今天的开放题是：把自然语言改成 JSON 输出要求：把下面这段需求改写成一个要求 AI 输出 JSON 的 Prompt，并定义至少 3 个字段：

「请帮我从一段餐厅评论中提取菜品名、评分、是否推荐。」

可以按这个模板来答：

1. 先写清楚角色或目标。
2. 再写输入是什么、处理步骤是什么。
3. 明确输出格式，不要让 AI 自由发挥。
4. 加上边界条件：不知道怎么办、失败怎么办、哪些不能做。
5. 最后检查评分标准：结构完整、细节具体、能直接拿去用。
