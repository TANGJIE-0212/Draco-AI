# Week 2 Day 1：角色扮演术：System Prompt 的魔力

## 🎬 开场（10-20 秒）

今天我们讲 Prompt 工程 里的「角色扮演术：System Prompt 的魔力」。这一课不要死记硬背，先抓住一个小白也能懂的主线：欢迎来到第二周：咒语魔法师。

这支视频必须覆盖当天所有题目，一共 10 个考点。看完以后，你不仅知道答案，还能说出为什么。

## 📖 主体讲稿

### Slide 1 — 先建立今天的直觉

把今天的内容想成一个工具箱。遇到题目时，先问三件事：它在解决什么问题？输入是什么？输出应该长什么样？只要这三件事想清楚，选择题、连线题和开放实战题都会变简单。

### Slide 2 — 考点 1-4：逐题白话拆解

- 当题目问「概念：在一个 ChatGPT 对话中，System Prompt 的作用是？」时，正确理解是：在幕后设定 AI 的身份、行为规则和输出风格。
- 当题目问「角色：当你在 API 调用中设置 role: 'system' 并写入「你是一个友善的客服代表」，这段话对谁生效？」时，正确理解是：对整个对话过程中 AI 的所有回复都生效。
- 当题目问「类比：System Prompt 就像导演给演员的 ___，决定了 AI 在整场对话中扮演什么角色。」时，正确理解是：剧本。
- 当题目问「设计：以下哪个 System Prompt 最专业？」时，正确理解是：你是一位资深数据分析师。你只使用用户提供的数据进行分析，不编造数据。你用要点列表呈现结论，每个结论附带数据支撑。。

讲的时候提醒学员：不要只背关键词，要把关键词放回真实场景里。题目一换皮，场景理解才是拿分关键。

### Slide 3 — 考点 5-8：逐题白话拆解

- 连线题要记成配对表：角色定义 = 你是一位经验丰富的儿科医生；能力边界 = 你只回答 0-12 岁儿童的健康问题；输出风格 = 用家长能听懂的语言回答，避免专业术语；禁忌事项 = 不做诊断，对于紧急情况建议立刻就医。
- 当题目问「反面教材：System Prompt 写「你是一个无所不能的 AI，可以回答任何问题」，最大的问题是？」时，正确理解是：过于宽泛，AI 容易在不擅长的领域编造答案（幻觉增加）。
- 开放实战要讲清楚：写一段角色描述：用 3-5 句话，给「英语口语陪练 AI」写一段角色和能力描述。说清楚：它是谁、能帮你做什么、不擅长做什么、回答时用什么语气。 评分时看这些点：是否包含角色身份、擅长的事、不擅长或拒绝的事、说话语气；是否针对“口语陪练”这个具体场景，而不是泛泛而谈；是否避免空话和夸张。
- 开放实战要讲清楚：修改一个不合格的 Prompt：下面这段 System Prompt 太宽泛，AI 容易乱说话：

「你是一个无所不能的 AI，可以回答任何问题。」

请你重写它，加上：1）一个具体角色；2）能力边界；3）一句不能做的事。 评分时看这些点：是否给出明确角色（不再是“无所不能”）；是否加了能力范围；是否加了至少一条禁止事项；是否比原版更不容易让 AI 编造答案。

讲的时候提醒学员：不要只背关键词，要把关键词放回真实场景里。题目一换皮，场景理解才是拿分关键。

### Slide 4 — 考点 9-10：逐题白话拆解

- 开放实战要讲清楚：迁移到你自己的场景：挑一个你真实想用 AI 帮忙的场景（比如做饭、健身、写周报、辅导孩子作业……），给它写一份 System Prompt 草稿。包含角色、能做什么、不做什么、说话风格。 评分时看这些点：场景是否真实具体；是否覆盖角色/能力/边界/语气四要素；是否能直接复制到 ChatGPT 使用；语言是否清晰不含废话。
- 开放实战要讲清楚：综合大题：请写一份完整、可直接复制使用的「儿童读物推荐 AI」System Prompt。要求包含：
1) 角色与目标；
2) 用户会提供的信息（如孩子年龄、兴趣）；
3) 推荐时的执行步骤；
4) 输出格式（书名 / 适龄段 / 推荐理由 / 注意事项）；
5) 至少 2 条边界限制。 评分时看这些点：是否覆盖以上 5 项；是否能直接拿去使用；是否避免编造书目（鼓励 AI 在不确定时说明）；是否考虑孩子年龄和家长可读性；是否结构清晰。

讲的时候提醒学员：不要只背关键词，要把关键词放回真实场景里。题目一换皮，场景理解才是拿分关键。

### Slide 5 — 小白最容易混淆的点

- 如果题目问“概念”，先给一句人话定义，再补英文术语。
- 如果题目问“场景”，先判断任务类型，再选工具或 Prompt 策略。
- 如果题目问“Debug”，先找根因，不要直接堆更多功能。
- 如果题目问“Boss 综合”，答案要有结构：目标、步骤、格式、边界、检查。

### Slide 6 — 本课实战模板

今天的开放题是：写一段角色描述：用 3-5 句话，给「英语口语陪练 AI」写一段角色和能力描述。说清楚：它是谁、能帮你做什么、不擅长做什么、回答时用什么语气。

可以按这个模板来答：

1. 先写清楚角色或目标。
2. 再写输入是什么、处理步骤是什么。
3. 明确输出格式，不要让 AI 自由发挥。
4. 加上边界条件：不知道怎么办、失败怎么办、哪些不能做。
5. 最后检查评分标准：结构完整、细节具体、能直接拿去用。
