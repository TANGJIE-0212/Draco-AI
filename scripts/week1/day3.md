## 📖 主体讲稿

### Slide 1 — 先建立今天的直觉

把今天的内容想成一个工具箱。遇到题目时，先问三件事：它在解决什么问题？输入是什么？输出应该长什么样？只要这三件事想清楚，选择题、连线题和开放实战题都会变简单。

### Slide 2 — 考点 1-4：逐题白话拆解

- 当题目问「概念热身：如果 AI 是一个乐高玩家，那么 Token 是什么？」时，正确理解是：最小的积木块。
- 当题目问「思维实验：AI 看到 'Strawberry' (草莓) 时，它实际接收到的输入更像以下哪种形式？」时，正确理解是：[8123, 4451] (两个代表音节的数字)。
- 当题目问「实战：数学盲区
数字 '9.11' 和 '9.8' 比较大小时，AI 常出错。因为 '9.11' 可能被切成 [9], [.], [11]，而 '9.8' 是 [9], [.], [8]。AI 会觉得 11 比 8 大。
这是因为 Tokenizer 破坏了数字的：」时，正确理解是：整体数值。
- 当题目问「我是老板：假设 API 价格是 $0.002 / 1k Tokens。你要处理一篇 5000 个单词的英文论文，大概需要支付多少钱？」时，正确理解是：约 $0.01 (5000 * 1.3 / 1000 * 0.002)。

讲的时候提醒学员：不要只背关键词，要把关键词放回真实场景里。题目一换皮，场景理解才是拿分关键。

### Slide 3 — 考点 5-8：逐题白话拆解

- 连线题要记成配对表：Tokenizer = 把文本切碎成 ID；Embedding = 把 ID 变成空间坐标；向量维度 = 描述语义的特征数量；空间距离 = 词义的相似程度。
- 当题目问「导航员测试：在 Embedding 星图中，'苹果' 这个词的坐标，应该离下面哪个词最近？」时，正确理解是：'香蕉'。
- 当题目问「语境感知：」时，正确理解是：科技/电子产品。
- 当题目问「炼金术士：请推测这个向量公式的结果：
Vector(巴黎) - Vector(法国) + Vector(日本) ≈ ？」时，正确理解是：东京。

讲的时候提醒学员：不要只背关键词，要把关键词放回真实场景里。题目一换皮，场景理解才是拿分关键。

### Slide 4 — 考点 9-9：逐题白话拆解
